# Bank Management System

A Bank Management System developed using Python and Streamlit that allows users to manage banking operations through a simple and interactive web interface.

## Features

* User Registration
* User Login
* Account Creation
* Deposit Money
* Withdraw Money
* Check Account Balance
* Secure Data Storage using JSON
* Interactive Streamlit Interface

## Technologies Used

* Python
* Streamlit
* JSON
* Object-Oriented Programming (OOP)

## How to Run the Project

1. Clone the repository:
   https://github.com/yashgeed-AI-DS/Bank-Management-System/new/main

2. Install dependencies:
   pip install -r requirements.txt

3. Run the application:
   streamlit run app.py
